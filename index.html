全等/不全等（相等/不相等）

相等和不相等操作符存在类型转换问题，而为了保证数据的完整性，我们一般使用全等和不全等操作符

函数参数

ECMAScript中也没有函数签名的概念，因为其函数参数是以一个包含多个零或多个值的数组的形式传递的。

可以向ECMAScript函数传递任意数量的参数，并且可以通过arguments对象来访问这些参数

argument是对象是个类数组（但是并不是Array的实例）

变量

ECMAScript变量可能包含两种不同的数据类型的值，一种是基本数据类型，另一种是引用数据类型

基本数据类型： symbol string number null undefined Boolean；

引用数据类型： object
